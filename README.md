chat-ai
Простой веб‑чат с бэкендом на FastAPI и фронтендом статического JS. Установите OPENAI_API_KEY в секретах репозитория и задеплойте бэкенд (например, Render).

После создания всех файлов — напишите "Готово" и я дам инструкции по добавлению секретов (Settings → Secrets) и развертыванию на Render с пошаговыми действиями на телефоне.*
